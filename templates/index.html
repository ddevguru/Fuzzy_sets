Fuzzy Set Operations

A short reference on how union, intersection, complement and the algebraic operators are defined on fuzzy sets — followed by a calculator that applies the same formulas to a set you enter, one element at a time.

What is a fuzzy set?

In an ordinary (crisp) set, an element either belongs or it doesn't. A fuzzy set relaxes this: every element x of the universe is assigned a membership value μ(x) between 0 and 1, describing the degree to which it belongs. A fuzzy set is written as

A = { μ₁/x₁ + μ₂/x₂ + … + μₙ/xₙ }
Each term pairs an element of the universe with its membership degree.

Standard operations

Given two fuzzy sets A and B on the same universe X, with membership functions μA(x) and μB(x):

Union  A ∪ B
μA∪B(x) = max{ μA(x), μB(x) }
Take the larger of the two membership values at each element.
Intersection  A ∩ B
μA∩B(x) = min{ μA(x), μB(x) }
Take the smaller of the two membership values at each element.
Complement  Ā
μĀ(x) = 1 − μA(x)
The degree to which x does not belong to A.
Difference  A | B
A|B = A ∩ B̄ = min{ μA(x), 1−μB(x) }
Intersection of A with the complement of B.
Algebraic sum  A + B
μA+B = [μA+μB] − [μA·μB]
Sum minus the overlap — keeps the result within [0, 1].
Algebraic product  A · B
μA·B(x) = μA(x) · μB(x)
Plain multiplication of the two membership values.
Bounded sum  A ⊕ B
μA⊕B(x) = min{ 1, μA(x)+μB(x) }
Ordinary sum, capped at 1 so it never overflows.
Bounded difference  A ⊖ B
μA⊙B(x) = max{ 0, μA(x)−μB(x) }
Ordinary difference, floored at 0 so it never goes negative.
